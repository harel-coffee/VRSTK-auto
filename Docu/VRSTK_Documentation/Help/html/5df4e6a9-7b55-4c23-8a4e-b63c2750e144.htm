Welcome to the Scientific Virtual Reality Toolkit

Welcome to the documentation for the Scientific Virtual Reality Toolkit. The Scientific Virtual Reality Toolkit (STK) simplifies the creation of experiments in VR environments. It was originally created as a master thesis by Felix Gähr at Hochschule Furtwangen.
